## 计算机如何取负值

如 -1

- 第一步取反码
  1 的 2 进制 32 位数 => 0000....1
  取反 => 1111....10

- 第二步补码(+1)
  1111....0 + 1 => 1111....1

相应的，一个负数的补码就是其对应的正数

- 取反码
  1111....1 => 0000....0

- 补码
  0000....0 + 1 => 0000....01

## js 的 >>> 运算

-1 >>> 0

1111....1111 

第一步，向右移动 0 位

 1111....1111 =>  1111....1111

第二步，符号位变为 0

1111....1111 => 0111....1111 

返回的是最大整数
